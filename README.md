# Discrete Markov Chain Simulation Project

This Python notebook (`.ipynb`) simulates a 2-state discrete Markov chain using the `numpy` and `matplotlib` libraries.

## Project Objectives:

- Define the transition matrix.
- Simulate a Markov chain trajectory.
- Count visits to each state.
- Calculate the theoretical distribution.
- Calculate the observed distribution.
- Compare results with graphs.

## Technologies Used:

- **Python**
- **NumPy** : For numerical calculations and matrix manipulation.
- **Matplotlib** : For visualizing results (trajectory, distributions).
